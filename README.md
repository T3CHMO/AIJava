# AIJava

This repository contains multiple Java-based examples demonstrating the use of AI and machine learning technologies, with a focus on integration with cloud AI services such as Azure OpenAI.

## Projects
- **Validation**: Spring Boot application showcasing validation of model responses in strict JSON format using Azure OpenAI chat models.

## Structure
Each subfolder contains a self-contained Java project. Refer to the individual project README files for details and setup instructions.

## Getting Started
1. Clone the repository:
   ```zsh
   git clone <repo-url>
   cd AIJava
   ```
2. Explore the available projects in the repository.
3. Follow the instructions in each project's README to build and run examples.

## Requirements
- Java 17 or higher
- Gradle
- (Optional) Azure OpenAI resource for cloud-based AI examples
